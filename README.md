Um comparador de preços de jogos digitais com foco em experiência moderna e performance
O projeto busca jogos em múltiplas lojas digitais, compara preços automaticamente e destaca a melhor oferta disponível.
---

# 🚀 Tecnologias

## Frontend

* React
* Vite
* TailwindCSS
* Framer Motion

## Backend

* Node.js
* Express
* Axios
* Fuse.js
* Node-Cache

## APIs Utilizadas

* Steam API
* CheapShark API
* OpenAI API (em integração)

---

# ✨ Funcionalidades

## 🔍 Busca de Jogos

* Busca em múltiplas lojas
* Resultados em tempo real
* Histórico de pesquisas
* Busca com Enter

## 💸 Comparação de Preços

* Comparação automática entre lojas
* Destaque do melhor preço
* Badge de desconto
* Ordenação do menor preço

## 🎨 Interface Moderna

* UI responsiva
* Skeleton loading
* Hover animations
* Gradientes e efeitos modernos

## ⚡ Performance

* Cache backend
* Requests paralelas
* Fuzzy matching inteligente

## 🛒 Redirecionamento para Compra

* Botão “Comprar”
* Links diretos para as lojas

# ⚙️ Instalação

## 1. Clone o repositório

```bash
git clone https://github.com/seuusuario/seurepositorio.git
```

---

# 🖥️ Frontend

## Entrar na pasta

```bash
cd frontend
```

## Instalar dependências

```bash
npm install
```

## Rodar projeto

```bash
npm run dev
```

---

# 🧠 Backend

## Entrar na pasta

```bash
cd backend
```

## Instalar dependências

```bash
npm install
```

## Rodar backend

```bash
npm run dev
```

---

# 🔑 Variáveis de Ambiente

Crie um arquivo `.env` dentro da pasta backend:
